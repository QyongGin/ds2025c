# 그래프
## 그래프 구조란?
버스 정류장과 여러 노선이 함께 포함된 형태 또는, 링크드인(Linked in)과 같은 사회 관계망 서비스의 연결 등의 형태
## 그래프의 개념
여러 노드가 서로 연결된 자료구조
## 그래프의 종류
